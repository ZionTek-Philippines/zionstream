## Publish & Migrate

```bash
# Spatie Permission
php artisan vendor:publish --provider="Spatie\Permission\PermissionServiceProvider"

# Filament panel (panel ID: zpanel)
php artisan filament:install --panels

# Reverb — interactive installer, needs app ID/key/secret input
# If running non-interactively, config and channels files are published automatically.
# Then set REVERB_* env vars manually in .env (see Environment Variables below).
php artisan reverb:install

# Livewire config
php artisan livewire:publish --config

# Run all migrations
php artisan migrate
```

## NPM Packages

```bash
# Runtime — install all together
npm install alpinejs @alpinejs/collapse @alpinejs/focus laravel-echo pusher-js agora-rtc-sdk-ng
```

> `alpinejs` must be installed as a separate npm package even though Livewire 4 bundles Alpine internally. Installing it explicitly lets us register Alpine plugins (`collapse`, `focus`) and wire up `window.Alpine` before Livewire loads.

> `pusher-js` is required by Laravel Echo even when using Reverb — it provides the client-side WebSocket protocol. No Pusher account or servers are used.

```bash
# Dev (already in package.json from project scaffold)

```

## Agora PHP Token Builder

Agora has no Composer package. Download the vendor files directly from the official repo:

```bash
mkdir -p app/Services/Agora/Vendor

curl -o app/Services/Agora/Vendor/RtcTokenBuilder2.php \
  https://raw.githubusercontent.com/AgoraIO/Tools/master/DynamicKey/AgoraDynamicKey/php/src/RtcTokenBuilder2.php

curl -o app/Services/Agora/Vendor/AccessToken2.php \
  https://raw.githubusercontent.com/AgoraIO/Tools/master/DynamicKey/AgoraDynamicKey/php/src/AccessToken2.php
```

Files land in `app/Services/Agora/Vendor/`. `RtcTokenBuilder2.php` requires `AccessToken2.php` via a relative `require_once`.

## Environment Variables

Add these to `.env` (values shown are local defaults):

```env
APP_NAME=ZionStream
APP_TIMEZONE=Asia/Manila

# Queue & Broadcasting
QUEUE_CONNECTION=redis
BROADCAST_CONNECTION=reverb

# Redis
REDIS_CLIENT=phpredis
REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379

# Reverb (generated by reverb:install)
REVERB_APP_ID=569901
REVERB_APP_KEY=your-reverb-key
REVERB_APP_SECRET=your-reverb-secret
REVERB_HOST="localhost"
REVERB_PORT=8080
REVERB_SCHEME=http

VITE_REVERB_APP_KEY="${REVERB_APP_KEY}"
VITE_REVERB_HOST="${REVERB_HOST}"
VITE_REVERB_PORT="${REVERB_PORT}"
VITE_REVERB_SCHEME="${REVERB_SCHEME}"

# Agora — get values from console.agora.io
AGORA_APP_ID=your-agora-app-id
AGORA_APP_CERTIFICATE=your-agora-app-certificate

VITE_AGORA_APP_ID="${AGORA_APP_ID}"
# ⚠️  AGORA_APP_CERTIFICATE is NEVER exposed via VITE_
```

## `resources/js/app.js`

```js
import Alpine from 'alpinejs';
import collapse from '@alpinejs/collapse';
import focus from '@alpinejs/focus';
import Echo from 'laravel-echo';
import Pusher from 'pusher-js';

window.Pusher = Pusher;

window.Echo = new Echo({
    broadcaster: 'reverb',
    key: import.meta.env.VITE_REVERB_APP_KEY,
    wsHost: import.meta.env.VITE_REVERB_HOST,
    wsPort: import.meta.env.VITE_REVERB_PORT,
    wssPort: import.meta.env.VITE_REVERB_PORT,
    forceTLS: (import.meta.env.VITE_REVERB_SCHEME ?? 'https') === 'https',
    enabledTransports: ['ws', 'wss'],
});

Alpine.plugin(collapse);
Alpine.plugin(focus);
window.Alpine = Alpine;
Alpine.start();
```
